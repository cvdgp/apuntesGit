Aprender a usar Git

¿Que es Git?

Git es un sistema de control de versiones que podemos utilizar para ir guardando las diferentes modificaciones que realizamos en nuestro código y en caso de error siempre poder regresas a una versión anterior. Un sistema de control de versiones nos permitirá que varios programadores trabajen al mismo tiempo y unifiquen su código en el mismo proyecto.

Git no es un lenguaje de programación. Se compone de una serie de comandos que utilizaremos desde una terminal para realizar una serie de acciones sobre las versiones de nuestro programa (crear una versión, guardarla, borrarla...).

¿Que un repositorio?

Es un espacio que dejamos para guardar nuestras actualizaciones. Cuando generamos un repositorio de forma automática git genera un grupo de carpetas y sub-carpetas donde almacenará la información de nuestro proyecto.

¿Como podemos crear un repositorio desde git CMD?

Abriremos la consola de git CMD y buscaremos la ruta donde deseamos dejar nuestro repositorio, para ello accederemos a la ruta utilizando el comando cd y la dirección donde deseamos dejar nuestro repositorio. ejemplo cd C:\wamp64\www\MisPaginasWeb\APUNTEShtml5.
En el caso de que deseamos retroceder una carpeta introduciremos cd, dos puntos y la barra cd .. /

Otra opción más rápida es acceder a la carpeta donde queremos iniciar un repositorio dar al botón derecho de nuestro ratón y seleccionar Git Bash Here, con ellos abriremos la consola cmd en la ruta en la que nos encontramos.

Una vez hemos elegido la ruta donde alojar nuestro repositorio introduciremos el comando git init. de esta forma git generará una carpeta con nuestro proyecto.

¿Que es un commit?

Son las diferentes modificaciones o versiones de nuestro programa guardadas dentro de un repositorio. Cuando trabajamos en git tenemos que acostumbrarnos a dar un commit a cada tarea que estamos realizando. Imagina que te encuentras realizando modificaciones en tu menú, en este caso seria conveniente te guardaras un commit nombrado con las modificaciones.

Para realizar un commit introduciremos el comando git commit, en ese momento se habilitara una nueva pantalla en la que debemos introducir el texto en el que resumimos nuestras modificaciones. luego bajaremos al final e introduciremos el comando :wq.

Otra opción es incluir en el commit el comentario git commit -m "comentario de la actualización"

Adjuntar un archivo en nuestro repositorio git add

Podemos incluir uno o varios archivo en nuestro repositorio. Utilizaremos el comando git add "nombreArchivo.html" realizando un git add por cada archivo o realizar un git add * con lo que incluiríamos todos los archivos.

¿Como podemos consultar histórico de commits realizados?

Cuando deseemos consultar los diferentes commits realizados sobre un repositorio utilizaremos el comando git log .
